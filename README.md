# MailQuell

## Gmail Project Setup

1. Navigate to the Google Developer Console.
2. Create a new project.
3. Enable the Gmail API for your project.
4. Create OAuth credentials and download the Client Secret JSON file.
5. Add the Client Secret file to your project directory.

## Configuration

- Update your application configuration with the Client Secret file path.
- Ensure you have the necessary scopes defined in your application for accessing Gmail API features.
